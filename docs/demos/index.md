# Demo autonomous AI Agents

Here you can find a demo AI agent that showcases the potential
of the {{open_autonomy}} framework. You can fetch the AI agent (and/or
the corresponding agent blueprint) from the remote IPFS registry and explore it by yourself.

- [Hello world AI agent](https://stack.olas.network/demos/hello-world/): a demo AI agent to showcase
  how to use the main components in an AI agent, without external interactions.
